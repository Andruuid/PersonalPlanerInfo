# Mitarbeiterplaner.info

Unabhängiger **Ratgeber zu Personalplanung** für Schweizer KMU – Dienstplan, Ferienplanung, Teamorganisation.

Produkt-Website: [easyplaner.ch](https://www.easyplaner.ch)

## Schnellstart

```powershell
npm install
npm run dev
```

## Konfiguration

```env
NEXT_PUBLIC_SITE_URL=https://www.mitarbeiterplaner.info
```

## Deployment

1. Repo auf GitHub: [PersonalPlanerInfo](https://github.com/Andruuid/PersonalPlanerInfo)
2. Vercel-Projekt verbinden
3. Domain `mitarbeiterplaner.info` hinzufügen
4. **GoDaddy-Weiterleitung auf easyplaner.ch entfernen** – diese Domain soll eigenständig laufen
5. Search Console: eigene Property für `mitarbeiterplaner.info`

## Verlinkung zu easyplaner.ch

Transparent im Footer und in Artikel-Callouts – keine versteckten Links.
